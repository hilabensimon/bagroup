# פרויקט בגרופ - למידה חברתית לבגרות

לומדה אינטראקטיבית (Web-based Learning Unit) שפותחה עבור משרד החינוך (מתודיקה). המערכת מספקת מידע על פרויקט הלמידה החברתית, מציגה עזרים פדגוגיים ומאפשרת למשתמשים למלא שאלון משוב. הלומדה מותאמת לתקן **SCORM** לצורך תקשורת מלאה עם מערכות ניהול למידה (LMS).

## 📖 אודות הפרויקט

המערכת בנויה כעמוד נחיתה (Single Page Application) הכולל שלושה חלקים עיקריים:
1. **אודות:** הסבר כללי על הפרויקט ומטרותיו.
2. **עזרים (Subjects):** הצגת כרטיסי מידע דינמיים המכילים משאבים למורים ולתלמידים.
3. **שאלון (Quiz):** טופס משוב הכולל שאלות אמריקאיות ושאלות פתוחות, עם מנגנוני בדיקה ושליחת נתונים.

## ✨ פיצ'רים מרכזיים

* **עיצוב רספונסיבי:** שימוש ב-Bootstrap 5 עם התאמה מלאה ל-RTL (ימין לשמאל) ולמובייל.
* **מנוע חיפוש פנימי:** סינון בזמן אמת (Live Search) בכרטיסי המידע, המאפשר איתור מהיר לפי שם או תיאור.
* **נגישות (Accessibility):** תמיכה ב-Screen Readers, ניווט מקלדת, וניהול פוקוס (Skip links, ARIA labels).
* **ולידציית טפסים:** בדיקת תקינות קלט מותאמת אישית (למשל: וידוא הזנת טקסט בעברית בלבד ובאורך מינימלי).
* **SCORM Wrapper:** עטיפה מלאה לתקשורת עם LMS, כולל שמירת סטטוס, ציונים ופירוט אינטראקציות.
* **מצבי קצה:** טיפול במצב של חוסר חיבור ל-LMS (למשל בהרצה מתוך GitHub Pages) עם הודעות מתאימות למשתמש.

## 🛠 טכנולוגיות

* **HTML5**
* **CSS3** (שימוש ב-Variables ו-Flexbox/Grid)
* **JavaScript** (ES6+)
* **Framework:** Bootstrap 5.2.3 (RTL version)

## 🚀 הוראות הרצה

### הרצה מקומית (פיתוח)
ניתן לפתוח את הקובץ `index.html` ישירות בדפדפן.
> **הערה:** בהרצה מקומית תופיע שגיאה בקונסול לגבי חוסר חיבור ל-LMS, והלומדה תעבוד במצב "Offline" (הודעות ייעודיות במודאל הסיום).

### העלאה ל-LMS (Production)
1. ודא כי קיים קובץ `imsmanifest.xml` בתיקייה הראשית (קובץ הגדרות החבילה).
2. כווץ את כל קבצי הפרויקט לקובץ **ZIP**.
3. העלה את ה-ZIP כרכיב SCORM למערכת הלמידה (Moodle, Totara, SAP וכו').
